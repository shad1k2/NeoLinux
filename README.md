# NeoLinux
NeoLinux — это пользовательская терминальная среда ( TUI-интерфейс рабочего стола ), построенная на основе Debian Linux. Она обеспечивает легковесный, поддерживаемый мышью «рабочий стол», полностью расположенный в терминале, без традиционных графических сред, таких как KDE или GNOME .

Обзор

NeoLinux пока не является полноценным дистрибутивом Linux , а представляет собой проект, который превращает минимальную установку Debian в продуктивное терминальное рабочее пространство. В основе лежит единственный скрипт Bash («textdesk.sh»), запускающий диалоговое меню с поддержкой мыши, интегрирующее популярные приложения TUI.

Цель — создать быструю, минималистичную и удобную систему, ориентированную на работу в терминале, для программирования, просмотра веб-страниц и выполнения повседневных задач — всё в текстовом режиме.

Функции

Пользовательский интерфейс рабочего стола TUI с кликабельным меню (с использованием диалогового окна).
Поддержка мыши в консоли (gpm + совместимые терминалы).
Предустановленные инструменты TUI:
* fastfetch — информация о системе с пользовательским ASCII-логотипом. 
 * mc и ranger — файловые менеджеры. 
 * htop — мониторинг процессов. 
 * mapscii — интерактивная ASCII-карта мира. 
 * w3m — текстовый веб-браузер. 
 * nmtui — настройка сети. 
 * tmux — мультиплексирование терминала.
Для простоты используйте doas вместо sudo .
Простая настройка — всё основано на скриптах.
Текущий статус
NeoLinux находится на ранней стадии разработки (версия 0.1 по состоянию на декабрь 2025 года). Это первый проект NeoLinux Project, в настоящее время представляющий собой набор скриптов и пакетов на основе минимальной версии Debian 13.

Планы на будущее:

Различные редакции (стабильная версия для Debian, Rolling для Arch, новая версия для Fedora).
Полный образ ISO с установщиком.
Независимая система управления посылками.
Установка
NeoLinux установлен поверх минимальной системы Debian: 1. Установите Debian 13 minimal/netinstall. 2. Запустите скрипт установки (скоро будет доступен на GitHub). 3. Перезагрузите компьютер и наслаждайтесь TUI-интерфейсом рабочего стола.
